# 插件

内置插件功能。 其实安装jgb-cli正常编译，引用jgb-weapp都是同一个实例的话，是没有必要使用插件功能的。 这里的插件功能是在没有引用jgb-weapp时，或者当引用jgb-weapp的实例不是同一个的情况。

# 使用

import { use } from 'jgb-weapp';

const plugin = {
  install(opts) {
    const { JApp, JPage, JComponent, jgb } = opts;

    JPage.mixin({
      // todo
    });
  }
};

use(plugin);
